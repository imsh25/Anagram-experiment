# AI Cognitive Offloading Experiment

## Overview
This is a web-based behavioral experiment investigating Cognitive Offloading and AI Dependency.

The study measures:
- AI usage behavior
- Reaction times
- Personality traits (EPQ)
- Correlation between AI usage and personality dimensions

## Technologies Used
- HTML
- Tailwind CSS
- JavaScript
- Google Apps Script (Data collection)
- GitHub Pages (Deployment)

## Research Goal
To examine whether personality traits predict reliance on AI assistance during problem-solving tasks.
